# Blog

A blog site focused on notes, references, and project documentation.

## Live Demo

A demo can be viewed ✨[HERE](https://blog.kazvee.com/)✨

## Features
* Toggleable light/dark theme
* Tags as taxonomy
* Atom RSS Feed
* Sitemap generation
* Non-post pages support (About page, etc.)

## Built With
* [Eleventy (11ty)](https://www.11ty.dev/)  
* [Stylus](https://stylus-lang.com/)  
* Vanilla JavaScript + small plugins (dark mode toggle)  
* [Eleventy RSS Plugin](https://www.11ty.dev/docs/plugins/rss/)  

## Installation
* Fork this repository to your own GitHub account
* Clone your fork onto your local device
* Install all dependencies with `npm i` (or `npm install`)
* Start the development site with `npm run dev`
* Build for deployment with `npm run build`
