This site uses [Umami](https://umami.is/) and [Cloudflare Web Analytics](https://www.cloudflare.com/products/web-analytics/) to understand how visitors interact with the site and improve the user experience.

## Umami
Runs in cookieless mode, storing no personal data.

## Cloudflare
Collects pageviews, referrers, and general location. Operates in cookieless mode and stores no personal information.
